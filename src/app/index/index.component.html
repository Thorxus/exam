Set A

1. How to create reactive forms?

Answer: To create reactive forms in Angular, you use the FormBuilder service. This service helps you construct a FormGroup, which is the core of a reactive form. You then define the form controls within the FormGroup using FormControl instances.

2. How to add built-in validation for email in reactive form?

Answer: To add built-in email validation, you use the email validator provided by Angular.

Example:
this.email = new FormControl('', [Validators.required, Validators.email]);

3. How to add built-in validation for required fields in reactive form?

Answer: To add the required field validator, use the required validator provided by Angular.

Example:
this.name = new FormControl('', Validators.required);

4. How to add regex validation for reactive forms

Answer: You can create custom validators using regular expressions (regex).

Example:
const customValidator = (control: AbstractControl): ValidationErrors | null => {
const regex = /^[a-zA-Z]+$/;
return regex.test(control.value) ? null : { 'nameInvalid': true }; };
this.name = new FormControl('', [Validators.required, customValidator]);

5. How to generate routing

Answer:
Using Angular CLI:
* ng generate component component-name
* ng generate module module-name
Manually:
* Create a new component (e.g., my-component.component.ts, my-component.component.html, my-component.component.css)
* Create a module (e.g., my-component.module.ts) and import the component into it.
* Configure the routing in your app-routing.module.ts file.

6. How to configure routes with parameters?

Answer:
* In the route definition:
{ path: 'user/:id', component: UserDetailComponent }
* In the component:
constructor(private route: ActivatedRoute) { }
ngOnInit() {
this.route.paramMap.subscribe(params => {
const userId = params.get('id');
});
}

7. What is DebounceTime in RxJS?

Answer: DebounceTime is an RxJS operator that suppresses the emission of values from a source Observable for a specified duration. It only emits the last value emitted by the source Observable within the duration. This is useful for preventing rapid emissions, such as in search input fields.

8. What is pipe operator?

Answer: The pipe operator in RxJS is used to chain multiple operators together to transform and manipulate the stream of data emitted by an Observable. It allows you to create complex data flows in a concise and readable manner.

9. What is distinctUntilChanged operator?

Answer: distinctUntilChanged is an RxJS operator that suppresses duplicate consecutive values emitted by the source Observable. It only emits a value if it is different from the previous emitted value.

10. What is filter operator in RxJs?

Answer: filter is an RxJS operator that selectively emits values from the source Observable based on a given predicate function. It only emits values that satisfy the condition defined in the predicate.
